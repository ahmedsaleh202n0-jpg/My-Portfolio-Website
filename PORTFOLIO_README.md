# Ahmed Saleh - Professional Portfolio

A modern, responsive, and beautifully designed portfolio website built with React, Tailwind CSS, and Framer Motion. This portfolio showcases my skills, projects, experience, and provides ways to get in touch.

## 🎯 Features

- ✨ **Modern Design**: Sleek dark theme with glassmorphism effects
- 🎬 **Smooth Animations**: Professional animations powered by Framer Motion
- 📱 **Fully Responsive**: Perfect display on mobile, tablet, and desktop devices
- 🎨 **Beautiful UI**: Custom gradient backgrounds and smooth hover effects
- ⚡ **Fast Performance**: Optimized with Vite for lightning-fast load times
- 🔄 **Smooth Scrolling**: Seamless navigation between sections
- 📊 **Interactive Skills Section**: Visual representation of technical skills
- 🎓 **Project Showcase**: Professional project cards with descriptions and links
- 📧 **Contact Form**: Built-in contact form with validation
- 🌐 **Social Links**: Direct links to GitHub, LinkedIn, and Email

## 🛠️ Tech Stack

- **React 18** - Modern UI library
- **Vite** - Next-generation frontend tooling
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Lucide React** - Beautiful icon library
- **JavaScript ES6+** - Modern JavaScript

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to:
```
http://localhost:5176
```

## 📦 Build for Production

```bash
npm run build
```

The optimized build will be in the `dist/` folder.

## 🎨 Customization

### Colors
Modify the color theme in `tailwind.config.js`:
```javascript
colors: {
  primary: '#7F5AF0',      // Main purple color
  secondary: '#2CB67D',    // Secondary green color
  dark: '#16161a',         // Dark background
  light: '#fffffe',        // Light text
}
```

### Personal Information
Update personal details in each section:
- **Hero Section**: Name, title, bio
- **About Section**: Personal description
- **Projects**: Project details and links
- **Contact Section**: Email, phone, social links
